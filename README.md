# Stopwatch Web Application

## 📌 Overview

The Stopwatch Web Application is a responsive and interactive timer built using HTML, CSS, and JavaScript. It allows users to accurately measure elapsed time with features such as Start, Pause, Reset, and Lap recording. The application offers a clean, user-friendly interface and works seamlessly across desktop and mobile devices.

---

## ✨ Features

- ▶️ Start the stopwatch
- ⏸️ Pause and resume timing
- 🔄 Reset the stopwatch
- 🏁 Record lap times
- 📱 Fully responsive design
- 🎨 Simple and modern user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Launch `index.html` in your web browser.

Or, use the **Live Server** extension in Visual Studio Code for the best development experience.
